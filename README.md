🎮 Tic Tac Toe AI

A modern Tic Tac Toe game with an AI opponent built using Python, featuring a clean UI and intelligent decision-making using the Minimax algorithm.

🚀 Features
🧠 AI opponent using Minimax Algorithm
🎯 Two difficulty levels:
Easy (Random moves)
Hard (Optimal AI)
🖥️ Interactive GUI built with Tkinter
🔄 Restart game functionality
🧮 Efficient board handling using NumPy
🎨 Clean and modern UI design
🧠 How the AI Works

The AI uses the Minimax Algorithm, which:

Simulates all possible game outcomes
Chooses the move that maximizes its chances of winning
Minimizes the opponent’s chances

👉 This makes the AI unbeatable in Hard mode

🎮 How to Play
You play as X
AI plays as O
Click on any empty cell to make your move
Choose difficulty:
Easy → Random AI
Hard → Smart AI (Minimax)
